Engine Data

Reference SFOC at any load: SFOC(L) = SFOC₁₀₀ × (a + b × (L/100)²). Enter SFOC @ 100% MCR (shop trial). Optional SFOC @ 85% calibrates the curve; otherwise a = 0.25, b = 0.75 (standard slow/medium-speed diesel).
Generator load % is calculated as (average kW ÷ max kW) × 100 for each alternator. Enter per-generator max kW from the nameplate.
Boiler and incinerator load % in supplementary reports uses the max kW values above.
Speed by engine (kn) = RPM × Pitch × 60 ÷ 1852. %MCR = (RPM ÷ 100%-Load RPM)³ × 100. KW = (RPM ÷ 100%-Load RPM)³ × 100%-Load KW — this is the basis used throughout Performance and the Reports' M/E figures.

Voyage Progress

Distance To Go, Days To Go and ETA are based on Total Voyage Distance (Setup → Voyage Specifics) minus cumulative distance run as of the last log entry, divided by that entry's own speed (falling back to the voyage average speed if the last entry has none).

Performance Averages

Fuel Efficiency (SFOC)

Reference SFOC uses SFOC₁₀₀ from Setup and the standard load curve SFOC(L) = SFOC₁₀₀ × (a + b × (L/100)²) with a + b = 1 (default a = 0.25, b = 0.75; calibrated through 85% when that shop-trial point is entered). Actual SFOC = M/E fuel (g) ÷ shaft energy (kWh) for each steaming period. A sustained +3% deviation from reference often warrants investigation (fouling, injection wear, turbo issues).

Voyage Time Averages

All averages are weighted by elapsed hours between log entries (not simple entry counts). Generator load % uses each alternator's max kW from Setup.

Server Sync

Configure your self-hosted sync server to back up and merge data when internet is available. Works offline — changes queue locally and upload on reconnect.
Last sync: never
Point the URL at your Linux server exposed via Cloudflare Tunnel (or nginx). Run sync-server/server.py on the server — see README. The app stores everything locally in IndexedDB and syncs JSON snapshots per vessel + voyage.

Local Database

All data (setup, every log entry, receipts) is stored automatically in this browser's local database (IndexedDB) — nothing is lost when you close the tab, reload, or reopen this file later in the same browser. Use Export as a portable backup or to move data to another browser/computer.

"Export Setup Only" saves just the Setup tab's configuration (vessel details, engine data, flowmeter arrangement, fuel/lube defaults, opening ROB, tank capacities) without any voyage log entries or receipts — handy as a reusable template for a new voyage. "Import Setup Only" loads just that configuration back in and leaves your current log entries and receipts untouched (unlike "Import Backup", which replaces everything).
Noon Report — offline-capable PWA with local IndexedDB storage and optional self-hosted sync. Install on Android or PC from the Data tab when served over HTTPS.
